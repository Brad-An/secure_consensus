# Secure Consensus

My Final Year Project to study privacy preserving consensus algorithms.

## Requirements

* Python 3
* NumPy
* SciPy
* Matplotlib
* gmpy2
* PyPrnt (My Library!)

## How to run

```bash
git clone [this repo]
cd secure_consensus
cd consensus_simulation
pipenv install
pipenv run python3 main.py
```
